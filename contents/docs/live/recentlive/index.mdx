## Guide
The `getRecentLive` function allows you to retrieve data about the most recent live streams from JKT48 members.  
This feature can be used in various projects, such as WhatsApp bots, Discord bots, and websites.  
Below is an example of how to use it:

<Note type="note" title="Note">
  This requires a premium API key. To get an API key, you can visit the [dashboard website](https://dash.jkt48connect.my.id) and purchase a premium key from the dashboard.
</Note>

```javascript:example.js showLineNumbers {1-13}
const { getRecentLive } = require("@jkt48connect-corp/sdk");

// API Key yang digunakan
const apiKey = "YOUR_API_KEY"; // Ganti dengan API Key yang valid

async function fetchRecentLiveData() {
    try {
        const recentLiveData = await getRecentLive(apiKey);
        console.log("Data Recent Live Streaming:", recentLiveData);
    } catch (error) {
        console.error("Gagal mengambil data recent live:", error.message);
    }
}

// Panggil fungsi untuk mendapatkan data recent live
fetchRecentLiveData();
```
