Setting up JKT48Connect is simple, with options to install the CLI module and access the REST API. Ensure you have [Node.js](https://nodejs.org/) version 18 or higher for optimal performance.

## Install JKT48Connect CLI
To use the JKT48Connect CLI, install it globally via npm:

<Stepper>
  <StepperItem title="Step 1: Install JKT48Connect CLI">
    Run the following command to install the CLI globally:

    ```plaintext
    npm install -g @jkt48connect/cli
    ```
  </StepperItem>

  <StepperItem title="Step 2: Verify Installation">
    Check if the CLI is installed correctly by running:

    ```plaintext
    jkt48connect 
    ```
  </StepperItem>

  <StepperItem title="Step 3: Authenticate">
    Before using the CLI, authenticate with your API key:

    ```plaintext
    jkt48connect {command} {apikey}
    ```

    <Note title="API Key Required">
      You must obtain an API key from the JKT48Connect platform to use this service.
    </Note>
  </StepperItem>
</Stepper>

## Using the REST API

JKT48Connect also provides a powerful REST API for integrating JKT48-related data into your applications.

### Base URL
All API requests should be made to the following base URL:

```plaintext
https://api.jkt48connect.my.id
```

### Example: Fetch Live Streaming Data
To fetch the latest live streaming data, send a GET request to:

```plaintext
GET /api/live?api_key=YOUR_API_KEY
```

**Response Example:**
```json
{
  "status": "success",
  "data": [
    {
      "member": "Shani Indira",
      "title": "Shani's Live",
      "startTime": "2025-03-03T12:00:00Z"
    }
  ]
}
```

## Quick Start

For a quick setup, follow these steps:

1. Install the CLI:
   ```plaintext
   npm install -g @jkt48connect/cli
   ```
2. Authenticate:
   ```plaintext
   jkt48connect 
   ```
3. Fetch live streaming data:
   ```plaintext
   jkt48connect live YOUR_API_KEY
   ```

With this setup, you can easily access JKT48Connect's features through both the CLI and REST API.
